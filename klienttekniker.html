Klienttekniker



HTML

HTML är ett märkspråk och det står för HyperText Markup Language. Ett märkspråk kommunicerar och beskriver för webbläsaren hur en hemsida vara uppbyggd och fungerar som ett skelett. HTML utgörs av taggar som definierar till exempel titlar, undertitlar och paragrafer.

CSS

CSS är ett stilmallsspråk som står för Cascading Style Sheet, och med hjälp av detta kan man bestämma utseendet på en hemsida. CSS kan användas tillsammans med HTML på två sätt, endera genom att använda style-taggen i HTML-filen, eller genom att länka till en separat CSS-fil. Det sistnämnda är det sätt som oftast rekommenderas, speciellt om det är ett större projekt.

JavaScript

JavaScript är ett skriptspråk och detta kan, till skillnad från HTML och CSS kategoriseras som ett programmeringsspråk eftersom det kan utföra kommandon. Med JavaScript kan man lägga till diverse funktioner när man till exempel trycker på en knapp på hemsidan.
