Internet



Klient-server-modellen

Klient-server-modellen beskriver hur servrar tillhandahåller tjänster till klienter, en klient kan till exempel vara en dator eller en smartphone. En enda server kan tillhandahålla resurser till flera klienter samtidigt. När en klient ansluter till en server kan detta endera accepteras eller avvisas av servern, om anslutningen accepteras, oftast genom att rätt lösenord har matats in, upprätthåller servern en anslutning med klienten via ett protokoll (se nedan).


Internetkommunikation

Internetkommunikation är ett nätverk av sammankopplade enheter som kan kommunicera med varandra. För att detta ska fungera krävs det att enheterna följer samma regelverk för att skicka och ta emot information, så kallade protokoll. Det mest grundläggande protokollet som används på internet är internetprotokollet (IP), alla internetanslutna enheter har en egen IP-adress som gör det möjligt att kunna kommunicera med andra enheter. För att förenkla användandet av IP-adresser använder internet-leverantörer så kallade DNS-servrar, som översätter det användaren skriver i webbläsarens adressfält till just IP-adresser.



Protokoll

Ett protokoll är ett regelverk som gör det möjligt för olika enheter att kommunicera med varandra, oavsett tillverkare eller typ. Olika protokoll används på olika sätt, exempelvis så kan en typ etablera kommunikationen medan en annan kan dirigera information från en enhet till en annan.


HTTP / HTTPS

HTTP är ett protokoll, som står för HyperText Transer Protocol, som styr hämtandet och skickandet av data mellan hemsidor och servrar. HTTP har sedan 2010-talet succesivt bytts ut mot en säkrare version, nämligen HTTPS, där s:et står för Secure och innebär att informationen som skickas och tas emot har krypterats för att skydda användare när de hanterar känslig information som till exempel bankärenden. Detta görs möjligt genom ett SSL-certifikat, vilket numera är ett krav för att rankas av Google.


FTP / FTPS

FTP står för File Transfer Protocol och är även det ett protokoll som hanterar kommunikation och används för att föra över filer. FTP har, precis som HTTP, en krypterad variant som kallas FTPS och även här står s:et för Secure. FTP-servrar är programvaruapplikationer som gör det möjligt att föra över filer från en enhet till en annan, servern fungerar som en mittpunkt mellan avsändarens och mottagarens enheter.
